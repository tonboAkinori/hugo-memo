awk

空白やタブで区切られたテキストを処理するコマンド。 演算機能もあり、プログラミング言語としても使用される。

基本構文

awk '
    BEGIN{
        最初に実行されるブロック
    }
    {
        メインブロック
    }
    END{
        最後に実行されるブロック
    }
    '

処理するファイルの内容は下記とする。

A   1 A
B   2 C
D   3 D
E   1 E

$ awk 'BEGIN{print "=== Start ==="}{print $0}END{print "=== End ==="}' test.txt
=== Start ===
A   1 A
B   2 C
D   3 D
E   1 E
=== End ===

スクリプトの読み込み

-f でスクリプトを指定する。

BEGIN{
    print "START"
}
{
    print "---" 
    print $0
}
END{
    print "END"
}

$ awk -f sf test.txt
START
---
A   1 A
---
B   2 C
---
D   3 D
---
E   1 E
END

制御構文

if-elseやandなども使える。

$ awk '{if($2==1) print $0}' test.txt
A   1 A
E   1 E
