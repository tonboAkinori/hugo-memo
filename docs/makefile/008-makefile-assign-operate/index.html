代入演算子

makeで使うことのできる演算子を記載する。

演算子 内容
= 変数が使われる度に評価される。シェルコマンドを代入している場合、変数を使う度に評価される。
:= 一度だけ評価するため、定数として使える。
?= 変数が定義されていないときのみ := として機能する
+= 加算代入演算子。
