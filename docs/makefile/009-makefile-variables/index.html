変数

自動変数 内容
CFLAGS Cコンパイラに与える追加のフラグ
CXXFLAGS C++コンパイラに与える追加のフラグ
CPPFLAGS Cプリプロセッサに与える追加フラグ、プログラムに使われる。
LDFLAGS -Lのようなリンカーを実行をサポートする時にコンパイラに与えられる追加フラグ。代わりにライブラリ(-lfoo)は LDLIBS 変数に追加するべき。
LDLIBS ライブラリフラグや名前をコンパイラに与える、リンカーが実行される時にサポートする。 -Lのようなライブラリではないリンカーフラグは LDFLAGS 変数に与える。
